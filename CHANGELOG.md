# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Open-source scaffolding: README, LICENSE, Code of Conduct, Contributing guide,
  Security policy, issue/PR templates.
- GitHub Actions workflows: CI, release, Dependabot.
- ESLint + Prettier + rustfmt + clippy configuration.

## [0.1.0] - 2026-01-01

### Added
- Initial public preview release.
- Cluster / Topic / Consumer Group / Schema Registry / Kafka Connect / ACL management.
- Message browsing, sending, importing and exporting.
- Cross-cluster topic data copy.
- SASL (PLAIN / SCRAM / OAUTHBEARER) and SSL support.
- English and 简体中文 localisation.
