# Kafka Desktop Manager 功能测试用例文档

> 适用范围：Tauri v2 + React（TypeScript）桌面端  
> 测试环境：Kafka `localhost:9092`（单节点、PLAINTEXT、无认证）  
> 已知 Topic：`DEV_DEMO_MESSAGE_SERVICE_ACTIVEUSER`、`DEV_DEMO_MESSAGE_SERVICE_BATCHCANCEL`、`DEV_AUDIT_LOG`、`DEV_TENANT_STATUS_CHANGED`、`__consumer_offsets`、`DEV_DEMO_MESSAGE_SERVICE_BUSINESSMESSAGE`、`DEV_TENANT_CREATED`、`DEV_TENANT_INIT_RESULT`

---

## 优先级说明

| 优先级 | 说明 |
|--------|------|
| **P0** | 阻断核心流程，必须通过 |
| **P1** | 重要功能，发布前建议全部通过 |
| **P2** | 增强与边缘场景 |

## TC-CONN：连接管理

### TC-CONN-001

| 项目 | 内容 |
|------|------|
| **标题** | 新建并保存 Kafka 连接 |
| **前置条件** | 应用已启动；本地 Kafka 可访问 |
| **步骤** | 1. 打开连接管理或新建连接入口<br>2. 填写名称、Bootstrap：`localhost:9092`<br>3. 保存连接 |
| **预期结果** | 连接列表中出现新记录；配置持久化（SQLite）；重启应用后仍存在 |
| **优先级** | P0 |

### TC-CONN-002

| 项目 | 内容 |
|------|------|
| **标题** | 编辑已保存连接 |
| **前置条件** | 至少存在一条已保存连接 |
| **步骤** | 1. 选择某条连接进入编辑<br>2. 修改名称或 Bootstrap 地址<br>3. 保存 |
| **预期结果** | 列表与详情显示更新后的信息；再次打开仍为修改后内容 |
| **优先级** | P1 |

### TC-CONN-003

| 项目 | 内容 |
|------|------|
| **标题** | 删除连接 |
| **前置条件** | 至少存在一条非当前必需连接（或可安全删除的测试连接） |
| **步骤** | 1. 在列表中选择连接<br>2. 执行删除并确认（若有二次确认） |
| **预期结果** | 连接从列表移除；数据库中记录删除；若该连接曾连接，状态应清理 |
| **优先级** | P1 |

### TC-CONN-004

| 项目 | 内容 |
|------|------|
| **标题** | 测试连接（成功） |
| **前置条件** | Kafka 在 `localhost:9092` 正常运行 |

| **预期结果** | 返回成功提示；无超时；后端 `test_connection` 未报错 |
| **优先级** | P0 |

### TC-CONN-005

| 项目 | 内容 |
|------|------|
| **标题** | 测试连接（失败：错误地址） |

### TC-CONN-006

| 项目 | 内容 |
